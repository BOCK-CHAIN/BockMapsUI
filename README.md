# Web Folder

This folder contains everything needed to run the Flutter app on web browsers.

## Contents

- `index.html` → Main HTML file for the app
- `favicon.png` → Website icon
- `manifest.json` → Web app metadata
- `assets/` → Static assets like images
- `main.dart.js` → Compiled Dart code for the browser (generated)

## Notes

- Modify `index.html` for custom HTML changes
- Add web-specific assets in `assets/`
- Configure web metadata in `manifest.json` if needed
